# 🙌 Get Start!

**1. 저장소를 클론 합니다.**

```shell
$ git clone https://github.com/yeomgahui/payhere-assignment.git
```

**2. 프로젝트 디렉토리로 이동합니다.**

```shell
$ cd payhear-assignment
```

**3. 필요한 패키지를 설치합니다.**

```shell
$ npm i 
```

**4. 개발 서버를 실행합니다.**

```shell
$ npm run start
```
